# GitHub Introduction

## GitHub Vocabulary
- git
- GitHub
- remote
- local
- branch
- clone
- repo (repository)
- GUI
- markdown

## GitHub Commands
- git status
- git add <file-name>
- git commit -m "meaningful message"
- git push origin main

## Keyboard Shortcuts
- command + s - save
- command + r - refresh
- command + x - cut
- command + c - copy
- command + v - paste
- command + z - undo
